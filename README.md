/ Experimentacion_2026
M/CF-2026 Clase de experimentación y métodos estadísticos
*Introduccion*
+ Durante el curso de Estadística Paramétrica aprendí a utilizar diversas herramientas para el análisis e interpretación de datos mediante el software R. Iniciamos con la organización y exploración de bases de datos, aplicando estadística descriptiva y elaborando gráficos para comprender mejor la distribución de la información. Posteriormente, trabajamos con pruebas de normalidad y homogeneidad de varianzas para verificar los supuestos necesarios en los análisis paramétricos. También aprendimos a evaluar relaciones entre variables mediante diferentes tipos de correlación, como Pearson, Spearman, Kendall y punto biserial, así como a desarrollar modelos de regresión lineal simple. Finalmente, realizamos comparaciones de medias mediante pruebas t para distintos escenarios. Además, durante el curso utilizamos GitHub como una herramienta para almacenar, organizar y dar seguimiento a nuestros scripts y actividades, lo que permitió llevar un mejor control de nuestro trabajo y familiarizarnos con herramientas de uso frecuente en la investigación científica.
